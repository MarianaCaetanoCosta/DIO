Como podemos definir “Estruturas de repetição”:

a estrutura permite a execução de uma sequência repetidamente, caso determinadas condições sejam satisfeitas.


int num = 5, count = 1;
        do {
            num += count;
            System.out.println(num);
        } while (count <= 3);
